## 실행 방법

1. 시스템은 사용자의 랜드마크를 활용하여 '고개 숙임', '고개 좌우 기울기', '어깨 높이 차', '어깨 말림',
  '상체 좌우 기울기'의 5가지 핵심 자세 지표를 실시간으로 측정한다.
2. 사용자가 '시작' 버튼을 누르면 '초기자세 설정' 모드로 진입한다.
   이 상태에서 사용자가 5가지 모든 지표가 '정상' 범위에 해당하는 바른 자세를 취하면,
   시스템이 이를 '초기 자세'로 자동 인식하여 '측정 중' 상태로 전환된다.
3. 측정된 값은 '정상', '경고', '위험' 3단계로 즉각 구분된다.
   이는 웹페이지 좌측 하단에 텍스트와 색상으로 표시됨과 동시에, 사용자 실루엣 외곽선 색상(초록/주황/빨강)이     위험도에 따라 변하며 직관적인 피드백을 제공한다.
4. 측정이 종료되면 결과 페이지로 이동한다.
   이 페이지는 전체 측정 시간 중 '올바른 자세', '경고 자세', '위험 자세'의 시간 비율을
   파이 차트로 요약하여 보여준다.
5. 또한, 5가지 각 지표별 측정 시간 동안 가장 자세가 나빴던 순간의 스크린샷과 해당 발생 시각,
  상태(정상/경고/위험)를 카드 형태로 시각화하여 제공한다.
  이를 통해 사용자는 어떤 자세가 가장 취약했는지 명확히 인지할 수 있다.
