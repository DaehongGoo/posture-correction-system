# 업무자 실시간 자세 평가 시스템

## 프로젝트 목적 및 소개

현대인의 장시간 컴퓨터 사용으로 인한 잘못된 자세 문제는 근골격계 질환을 유발하는 주요 원인이다. 특히 사무실 및 재택근무 환경에서 발생하는 고개 숙임, 고개 기울기, 어깨 높이 차, 상체 좌우 기울기, 어깨 말림 등의 비정상적인 자세가 문제이다. 본 프로젝트는 MediaPipe의 Pose 인식을 활용해 5가지 주요 자세 지표를 실시간으로 분석하는 알고리즘을 개발하였다. 또한, Flask 웹 프레임워크와 HTML 인터페이스를 통해 사용자가 웹캠 화면으로 자신의 자세를 직접 확인하고, '정상', '경고', '위험' 3단계의 직관적인 시각적 피드백을 받아 올바른 자세를 유지하도록 돕는 실시간 모니터링 시스템을 구현하는 것을 목적으로 한다

## 기능
1. 웹캠 기반 실시간 포즈(랜드마크) 추적
- 웹캠 입력을 받아 MediaPipe Pose로 상반신 랜드마크(관절 포인트)를 실시간 추적합니다.
- 추적된 좌표를 기반으로 자세 지표를 계산하고 상태를 판단합니다.

2. 5가지 자세 지표 실시간 측정

본 시스템은 다음 5개 항목을 핵심 지표로 측정합니다.
- 고개 상하 기울기(고개 숙임)
- 고개 좌우 기울기
- 상체 좌우 기울기
- 어깨 좌우 높이 차
- 어깨 말림(rounded shoulder)

3. 초기 자세(기준 자세) 설정 → 사용자 맞춤 보정

사용자가 시작 버튼을 누르면 초기 자세 설정 단계로 진입합니다.

사용자의 5가지 자세 지표가 '정상'이 되면 순간 화면에 표시되는 자세를 기준 자세로 저장하고 화면에 회색의 라인으로 고정하여 올바른 자세의 가이드 라인을 제시합니다.

4. 위험도 3단계 피드백(정상/경고/위험) 시각화

측정 결과를 정상 / 경고 / 위험으로 구분하여 실시간 표시합니다
- 텍스트 및 색상 변화(실루엣/외곽선 등)로 위험도를 직관적으로 전달해, 사용자가 “지금 자세가 어떤 상태인지” 즉시 인지할 수 있습니다.
*분류 로직은 의학 가이드 라인 참고

5. 측정 종료 후 결과 리포트 제공

측정이 종료되면 결과 페이지에서 아래 내용을 제공합니다.

- 자세 상태 비율 요약(파이 차트)
  - 전체 측정 시간 중 정상/경고/위험이 차지한 비율을 시각화

- 최악 자세 순간 자동 기록 카드
  - 5개 지표 각각에 대해 측정 중 가장 나빴던 순간의 스크린샷 + 발생 시각 + 당시 상태를 카드 형태로 제공
  → 사용자가 자신의 취약한 자세 패턴을 명확히 확인 가능

## 실행 방법

1. 시스템은 사용자의 랜드마크를 활용하여 '고개 숙임', '고개 좌우 기울기', '어깨 높이 차', '어깨 말림', '상체 좌우 기울기'의 5가지 핵심 자세 지표를 실시간으로 측정한다.
2. 사용자가 '시작' 버튼을 누르면 '초기자세 설정' 모드로 진입한다.
이 상태에서 사용자가 5가지 모든 지표가 '정상' 범위에 해당하는 바른 자세를 취하면, 시스템이 이를 '초기 자세'로 자동 인식하여 '측정 중' 상태로 전환된다.
3. 측정된 값은 '정상', '경고', '위험' 3단계로 즉각 구분된다.
이는 웹페이지 좌측 하단에 텍스트와 색상으로 표시됨과 동시에, 사용자 실루엣 외곽선 색상(초록/주황/빨강)이 위험도에 따라 변하며 직관적인 피드백을 제공한다.
4. 측정이 종료되면 결과 페이지로 이동한다.
이 페이지는 전체 측정 시간 중 '올바른 자세', '경고 자세', '위험 자세'의 시간 비율을 파이 차트로 요약하여 보여준다.
5. 또한, 5가지 각 지표별 측정 시간 동안 가장 자세가 나빴던 순간의 스크린샷과 해당 발생 시각, 상태(정상/경고/위험)를 카드 형태로 시각화하여 제공한다.
이를 통해 사용자는 어떤 자세가 가장 취약했는지 명확히 인지할 수 있다.

## 역할
자세 평가 기준을 만들기 위한 의학적 근거 및 각도 거리 지표 리서치와 Baseline 기반 초기 자세 설정 UX 설계, 그리고 캡처 기반 직관 피드백 기능 아이디어를 주도적으로 제안하고 요구사항으로 구체화해 프로젝트의 제품 설계 방향을 만든 역할을 담당하였다.

## 보완
- 경고 알림 및 미니 팝업 기능 확장
화면 오른쪽 하단에 작은 팝업창 형태의 미니 UI를 추가하여, 사용자가 다른 작업을 하는 중에도 자세 상태(정상·경고·위험)를 간편히 확인할 수 있도록 개선할 수 있다.
- 개인별 데이터베이스 구축 및 피드백 분석
측정 결과를 개인별 DB로 저장함으로써 사용자의 자세 변화 추이를 분석하고, 주기적인 피드백 및 개선 리포트를 제공할 수 있다. 이를 통해 장기적인 자세 관리 및 습관 형성이 가능하다.
- 다양한 직업군 맞춤 적용
학생, 콜센터 직원, 사무직 종사자 등 장시간 앉아있는 직군에 우선 적용할 수 있으며, 향후 직업군별 업무 환경에 따라 ‘올바른 자세 기준’을 세분화해 맞춤형 교정이 가능하도록 발전시킬 수 있다.
